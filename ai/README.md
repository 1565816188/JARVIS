# ai

## 技术栈

- 构建 API 框架[fastapi](https://fastapi.tiangolo.com/zh/)

## 环境

- [python](https://www.python.org/downloads/)

## 安装

```shell
pip install pipreqs
```

## 在当前目录生成

```shell
pipreqs . --encoding=utf8 --force
```

## 包安装

```shell
pip install -r requirements.txt
```

## 启动命令行

```shell
python main.py
```
